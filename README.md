## Стадия разработки
На данный момент проект находится в начальной стадии разработки.

## Сборка
* На Linux:
1. cmake CMakeLists.txt -B build -S ${PWD}
2. make -C build -j4

* На Windows:
1. cmake CMakeLists.txt -G "MinGW Makefiles" -B build -S %cd%
2. mingw32-make -C build -j4
